# Phase 2 Goal - Agency standards and audit rules

## Status

Upcoming

## Goal

Define the agency standards and audit rules that teach the GTM Container Audit & Patch Package workflow what "good" looks like.

## Outcome

The project has stable reference standards for naming, consent/privacy review, GTM audit rules, and human-readable output writing.

## Included tasks

- Task 2.1 - Create agency naming conventions
- Task 2.2 - Create consent and privacy review standard
- Task 2.3 - Create GTM audit rule catalog
- Task 2.4 - Create output writing standard

## Out of scope

- Running audits.
- Creating optimized GTM JSON.
- Building scripts or validators.
- Configuring live API access.
- Claiming legal, privacy, or production safety certification.
