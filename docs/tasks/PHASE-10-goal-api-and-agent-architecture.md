# Phase 10 Goal - Move toward API and agent architecture

## Status

Future

## Goal

Prepare the MLP system to grow into API-based structured-output runners, read-only connectors, multi-agent orchestration, and approval-gated pre-production GTM writes.

## Outcome

Future architecture preserves the GTM Container Audit & Patch Package contract while adding controlled API and agent capabilities.

## Included tasks

- Task 10.1 - Add Responses API structured-output runner
- Task 10.2 - Add read-only GTM API connector
- Task 10.3 - Add read-only GA4 Data API connector
- Task 10.4 - Add read-only Google Ads reporting connector
- Task 10.5 - Introduce Agents SDK orchestration
- Task 10.6 - Add approval-gated pre-production GTM writes

## Out of scope

- MLP file-in/file-out launch.
- Unapproved writes.
- Publishing GTM versions.
- Broad production automation without guardrails.
