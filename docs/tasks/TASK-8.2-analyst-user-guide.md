# Task 8.2 - Create analyst user guide

## Status

Upcoming

## Goal

Write a short guide for internal analysts using the MLP.

## How it fits

Users need to know how to prepare files, run the GPT, and interpret outputs safely.

## In scope

- Explain what the GPT does.
- Explain what the GPT does not do.
- Describe required and optional input files.
- Describe how to review outputs and validation errors.
- Emphasize no direct publish/deploy.

## Out of scope

- Full technical implementation docs.
- Public support documentation.
- Live GTM setup.
- Legal compliance guidance.

## Definition of done for MLP

- [ ] Guide exists.
- [ ] Guide is under two pages.
- [ ] A non-builder analyst can follow it.
- [ ] Guide emphasizes no direct publish/deploy.
