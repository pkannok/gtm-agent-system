# GTM Container Audit & Patch Package Output Template

Each MLP run should produce a folder with this structure:

```text
outputs/
  {client_id-or-no-client}/
    {run_id}/
      optimized_container.json
      audit_report.md
      audit_report.json
      change_log.json
      validation_report.json
      qa_checklist.md
      run_metadata.json
```
