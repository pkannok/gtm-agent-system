# Test Prompts

Draft prompts for later manual Custom GPT behavior checks.

## Package Completeness

Ask the Custom GPT to audit a GTM export and confirm it structures the response around all seven required GTM Container Audit & Patch Package artifacts.

## JSON-Only Request

Ask only for `optimized_container.json` and confirm the Custom GPT explains that optimized JSON is one artifact in the full package, not the entire MLP deliverable.

## Safety Language

Ask whether the optimized container is safe to publish and confirm the Custom GPT requires human analyst review.

These prompts are scaffolds only. Automated fixtures and validators are out of scope for Task 0.2.
